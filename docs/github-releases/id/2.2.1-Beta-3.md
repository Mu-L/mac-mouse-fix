# 2.2.1 Beta 3
***Tanggal Rilis:** 6 Sep 2022*

<br>

Mac Mouse Fix __2.2.1 Beta 3__ mengembalikan kompatibilitas dengan versi macOS yang lebih lama & lainnya.

- Memperbaiki crash saat peluncuran di versi macOS yang lebih lama. Seharusnya menyelesaikan Masalah [#327](https://github.com/noah-nuebling/mac-mouse-fix/issues/327).
- Memperbaiki crash saat mencoba menambahkan aplikasi yang tidak memiliki nama aplikasi ke Pengaturan Khusus Aplikasi. Seharusnya menyelesaikan Masalah [#241](https://github.com/noah-nuebling/mac-mouse-fix/issues/241). Terima kasih khusus kepada [jeongtae](https://github.com/jeongtae) yang sangat membantu dalam menemukan masalahnya!
- Lebih banyak perbaikan bug kecil dan peningkatan di balik layar.
