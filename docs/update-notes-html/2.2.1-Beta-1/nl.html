Ventura-ondersteuning!

Mac Mouse Fix ondersteunt nu volledig macOS 13 Ventura. Speciale dank aan @chamburr die hielp met Ventura-ondersteuning in GitHub Issue #297.

Ondersteuning voor oudere macOS-versies vervallen

Helaas staat Apple alleen toe om voor macOS 10.13 High Sierra en later te ontwikkelen wanneer je vanaf macOS 13 Ventura ontwikkelt.

De minimaal ondersteunde versie is daarom verhoogd van 10.11 El Capitan naar 10.13 High Sierra.

Het spijt me hiervan. Maar om je op te vrolijken is er een relaxte lieveheersbeestje in de volgende sectie.

Bugfixes 🐞

  • Een crash in ‘App-specifieke instellingen’ opgelost bij het toevoegen van een app zonder ‘Bundle ID’. Kan helpen met GitHub Issues #289 en #241.
  • Een probleem opgelost waarbij Mac Mouse Fix het scrollgedrag van sommige tekentabletten veranderde. Zie GitHub Issue #249.
  • Een probleem opgelost waarbij sneltoetsen met de ‘A’-toets niet konden worden opgenomen. Lost GitHub Issue #275 op.
  • Een probleem opgelost waarbij sommige knoptoewijzingen niet goed werkten bij gebruik van een niet-standaard toetsenbordindeling.
  • Andere kleine fixes en visuele verbeteringen.
