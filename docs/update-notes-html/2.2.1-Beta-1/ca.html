Suport per a Ventura!

Mac Mouse Fix ara és totalment compatible amb macOS 13 Ventura. Agraïments especials a @chamburr que va ajudar amb el suport per a Ventura a l’Issue de GitHub #297.

S’ha eliminat el suport per a versions anteriors de macOS

Malauradament, Apple només permet desenvolupar per a macOS 10.13 High Sierra i posteriors quan es desenvolupa des de macOS 13 Ventura.

Per tant, la versió mínima compatible ha augmentat de 10.11 El Capitan a 10.13 High Sierra.

Em sap greu això. Però per animar-vos hi ha una marieta relaxada a la següent secció.

Correccions d’errors 🐞

  • S’ha corregit un error a ‘Configuració específica per aplicacions’ quan s’intentava afegir una aplicació sense ‘Bundle ID’. Podria ajudar amb els Issues de GitHub #289 i #241.
  • S’ha corregit un problema on Mac Mouse Fix canviava el comportament del desplaçament d’algunes tauletes de dibuix. Vegeu l’Issue de GitHub #249.
  • S’ha corregit un problema on les dreceres de teclat que incloïen la tecla ‘A’ no es podien enregistrar. Corregeix l’Issue de GitHub #275.
  • S’ha corregit un problema on algunes reassignacions de botons no funcionaven correctament quan s’utilitzava una distribució de teclat no estàndard.
  • Altres petites correccions i millores visuals.
