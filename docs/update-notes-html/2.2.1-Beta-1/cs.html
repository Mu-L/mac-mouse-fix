Podpora Ventury!

Mac Mouse Fix nyní plně podporuje macOS 13 Ventura. Speciální poděkování @chamburr, který pomohl s podporou Ventury v GitHub Issue #297.

Ukončení podpory starších verzí macOS

Bohužel, Apple umožňuje vývoj pro macOS 10.13 High Sierra a novější pouze při vývoji z macOS 13 Ventura.

Takže minimální podporovaná verze se zvýšila z 10.11 El Capitan na 10.13 High Sierra.

Je mi to líto. Ale pro zlepšení nálady je v další sekci pohodová beruška.

Opravy chyb 🐞

  • Opravena chyba v ‘Nastavení pro jednotlivé aplikace’ při pokusu o přidání aplikace bez ‘Bundle ID’. Může pomoct s GitHub Issues #289 a #241.
  • Opravena chyba, kdy Mac Mouse Fix měnil chování scrollování u některých grafických tabletů. Viz GitHub Issue #249.
  • Opravena chyba, kdy nebylo možné zaznamenat klávesové zkratky obsahující klávesu ‘A’. Opravuje GitHub Issue #275.
  • Opravena chyba, kdy některá přemapování tlačítek nefungovala správně při použití nestandardního rozložení klávesnice.
