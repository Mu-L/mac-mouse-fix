Ventura-Unterstützung!

Mac Mouse Fix unterstützt jetzt vollständig macOS 13 Ventura. Besonderer Dank geht an @chamburr, der bei der Ventura-Unterstützung in GitHub Issue #297 geholfen hat.

Einstellung der Unterstützung für ältere macOS-Versionen

Leider erlaubt Apple nur die Entwicklung für macOS 10.13 High Sierra und neuer, wenn man von macOS 13 Ventura aus entwickelt.

Die minimal unterstützte Version wurde daher von 10.11 El Capitan auf 10.13 High Sierra angehoben.

Das tut mir leid. Aber zur Aufmunterung gibt es im nächsten Abschnitt einen entspannten Marienkäfer.

Fehlerbehebungen 🐞

  • Ein Absturz in ‘App-spezifische Einstellungen’ wurde behoben, der beim Hinzufügen einer App ohne ‘Bundle ID’ auftrat. Könnte bei den GitHub Issues #289 und #241 helfen.
  • Ein Problem wurde behoben, bei dem Mac Mouse Fix das Scrollverhalten einiger Grafiktabletts veränderte. Siehe GitHub Issue #249.
  • Ein Problem wurde behoben, bei dem Tastaturkurzbefehle mit der Taste ‘A’ nicht aufgezeichnet werden konnten. Behebt GitHub Issue #275.
  • Ein Problem wurde behoben, bei dem einige Tasten-Neubelegungen mit nicht-standardmäßigen Tastaturlayouts nicht richtig funktionierten.
  • Weitere kleine Korrekturen und visuelle Verbesserungen.
