ℹ️ एआई द्वारा अनुवादित

यह रिलीज़ नोट क्लॉड एआई द्वारा अनुवादित किया गया है। इसमें त्रुटियाँ हो सकती हैं।
मूल अंग्रेजी संस्करण के लिए, यहाँ क्लिक करें।

Mac Mouse Fix 2 में पेश की गई शानदार चीज़ों को भी देखें!

ऐप फिर से काम कर रहा है।

मैंने ऐप को एक नए सर्टिफिकेट के साथ साइन किया है। पुराना सर्टिफिकेट Apple द्वारा रद्द कर दिया गया था जिससे Mac Mouse Fix के पुराने वर्जन अनुपयोगी हो गए थे। अधिक जानकारी के लिए इस पोस्ट को देखें।

नई सुविधा!

अब आप अपने माउस से सीधे मीडिया प्लेबैक, स्पीकर वॉल्यूम, डिस्प्ले ब्राइटनेस और बहुत कुछ नियंत्रित कर सकते हैं! यह संभव है क्योंकि Mac Mouse Fix अब आपको अपने माउस बटन को कीबोर्ड की किसी भी कुंजी से रीमैप करने देता है - यहां तक कि Apple के प्रोप्राइटरी फंक्शन कीज़ जैसे “प्ले-पॉज़” या “म्यूट स्पीकर” से भी।
