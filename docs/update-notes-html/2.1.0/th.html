ผมได้เซ็นแอพด้วยใบรับรองใหม่ ใบรับรองเก่าถูกเพิกถอนโดย Apple ซึ่งทำให้ Mac Mouse Fix เวอร์ชันเก่าใช้งานไม่ได้ ดูข้อมูลเพิ่มเติมได้ที่ โพสต์นี้

ฟีเจอร์ใหม่!

ตอนนี้คุณสามารถควบคุม การเล่นสื่อ, ระดับเสียงลำโพง, ความสว่างหน้าจอ และอื่นๆ ได้โดยตรงจากเมาส์ของคุณ! เป็นไปได้เพราะ Mac Mouse Fix ตอนนี้ให้คุณกำหนดปุ่มเมาส์ใหม่เป็นปุ่มใดก็ได้บนคีย์บอร์ด - แม้แต่ปุ่มฟังก์ชันพิเศษของ Apple อย่าง “เล่น-หยุด” หรือ “ปิดเสียงลำโพง”
