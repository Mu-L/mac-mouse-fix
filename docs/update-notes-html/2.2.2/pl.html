Problemy z dostępnością

2.2.2 rozwiązuje problem, w którym “Mac Mouse Fix Helper” nie mógł uzyskać Dostępu do Dostępności w Ustawieniach systemowych po aktualizacji z Mac Mouse Fix 2.2.0 lub wcześniejszej. Zobacz Problem na GitHub #412.

Poprawki błędów

  • Naprawiono problem, gdzie powiadomienia znikały gwałtownie zamiast płynnie zanikać z animacją w macOS Ventura
